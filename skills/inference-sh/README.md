# inference.sh AI 云平台工具

> Hermes Agent 技能：通过 inference.sh 平台调用 150+ 个 AI 服务

## 概述

一次注册、一个 API Key，即可使用 150+ 个 AI 服务。无需为每个服务商单独管理 API 密钥。

## 覆盖的服务

| 类别 | 服务 |
|------|------|
| 🎨 AI 绘图 | FLUX、Reve、Seedream、Grok Imagine、Gemini |
| 🎬 视频生成 | Veo、Wan、Seedance、OmniHuman、HunyuanVideo |
| 🤖 大语言模型 | Claude、Gemini、Kimi、GLM-4 |
| 🔍 搜索引擎 | Tavily、Exa |
| 🏗️ 3D 生成 | Rodin |
| 🐦 社交媒体 | Twitter/X 自动化 |
| 🎵 音频 | TTS、语音克隆 |

## 使用方式

对 Agent 说类似以下的话：

```
用 inference.sh 帮我生成一张图片
调用 FLUX 画一只猫
帮我查一下 inference.sh 上有哪些视频生成模型
```
