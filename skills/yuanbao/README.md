# 元宝群组交互工具

> Hermes Agent 技能：在元宝（Yuanbao）群组中 @提及用户、发送私信、查询群信息

## 概述

该技能让 Hermes Agent 能够与元宝（腾讯元宝）群组进行交互。Agent 可以：
- **@提及（艾特）** 群成员 — 直接回复即可自动转换
- **发送私信（DM）** 给特定用户
- **查询群信息** — 群名称、群主、成员数
- **查询群成员** — 搜索用户、列出机器人/助手

## 使用方式

对 Agent 说类似以下的话即可触发：

```
帮我艾特张三
@元宝 在吗？
查一下这个群有多少人
给小明发私信说 "晚上见"
```

Agent 会自动调用工具完成操作。

## @提及（艾特）说明

这是最常用的功能。你只需要在回复中包含 `@昵称`：

1. Agent 会先查询群成员获取准确的昵称
2. 然后在回复中使用 `@正确昵称`
3. 元宝网关自动将 `@昵称` 转换为真正的 @提及通知

**不需要特殊操作** — Agent 的回复文本就是最终发送的消息。

## 私信功能

| 参数 | 说明 |
|------|------|
| 群编号 | 从当前聊天中自动提取 |
| 目标用户 | 按昵称搜索，自动查找 |
| 消息内容 | 你要发送的文字 |
| 媒体文件 | 支持图片（jpg/png/gif/webp/bmp）和文档 |

## 技术细节

- `group_code` 从 chat_id 提取：`group:328306697` → `328306697`
- 元宝中群组称为"派（Pai）"
- 成员角色：普通用户（`user`）、AI 助手（`yuanbao_ai`）、机器人（`bot`）
