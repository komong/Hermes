# Dogfood Web QA 自动化测试工具

> Hermes Agent 技能：对 Web 应用进行系统化的探索性 QA 测试，自动发现 Bug 并生成报告

## 概述

Dogfood 是一个全自动的 Web 应用 QA 测试工具。Agent 会像真实用户一样浏览你的网站，点击按钮、填写表单、滚动页面，检查每个环节是否存在问题，最后生成一份结构化的 Bug 报告。

**它能发现什么：**
- 🔴 功能缺陷 — 按钮没反应、表单提交失败、导航跳转错误
- 🎨 视觉问题 — 布局错位、图片加载失败、样式不一致
- 🖥️ 控制台错误 — JS 异常、网络请求失败、API 报错
- ♿ 无障碍问题 — 缺少 alt 文本、对比度不足、键盘导航失效
- 🧩 体验问题 — 缺少加载提示、交互反馈缺失、错误信息不清晰

## 使用方法

对 Agent 说类似以下的话即可触发：

```
帮我测试一下这个网站：https://example.com
对我的网站做个全面的 QA 测试
检查一下这个页面上有没有 Bug
```

Agent 会执行 5 阶段工作流：

1. **规划阶段** — 分析页面结构，制定测试计划
2. **探索阶段** — 逐个页面浏览，点击、填写、滚动
3. **收集证据** — 发现问题后截图 + 记录重现步骤
4. **分类整理** — 按严重程度和类型归类
5. **生成报告** — 输出结构化的 Bug 报告

## 严重等级

| 等级 | 说明 |
|------|------|
| 🔴 Critical | 核心功能不可用或数据丢失 — 崩溃、白屏、登录失败 |
| 🟠 High | 功能严重受损但可能有替代方案 |
| 🟡 Medium | 影响体验但不阻塞核心功能 |
| 🔵 Low | 小瑕疵 — 错别字、间距问题、占位文本 |
