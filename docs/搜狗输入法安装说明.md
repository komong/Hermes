# 搜狗输入法 Ubuntu 22.04 安装说明

> 适用于 Ubuntu 22.04 LTS x86_64 + X11 会话

---

## 环境确认

在开始前，确认你的系统环境：

```bash
# 查看系统版本
cat /etc/os-release

# 查看会话类型（应为 x11）
echo $XDG_SESSION_TYPE

# 查看架构
uname -m
# 预期输出: x86_64
```

- ✅ 适合系统: Ubuntu 22.04 LTS
- ⚠️ Ubuntu 24.04: 兼容性问题较多，不推荐
- ⚠️ Wayland 会话: 搜狗与 Wayland 不兼容，建议切换到 X11

---

## 安装步骤

### 第一步：更新系统

```bash
sudo apt update
```

### 第二步：安装 fcitx4 输入法框架

```bash
sudo apt install fcitx fcitx-bin fcitx-table
```

**配置环境变量**（写入 `~/.xprofile`，确保 X11 会话加载）：

```bash
cat >> ~/.xprofile << 'EOF'

# Fcitx input method
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"
EOF
```

**设置 im-config 为 fcitx**：

```bash
im-config -n fcitx
```

**设置 fcitx 开机自启**：

```bash
sudo cp /usr/share/applications/fcitx.desktop /etc/xdg/autostart/
```

### 第三步：安装 Qt5 核心依赖

```bash
sudo apt install libqt5qml5 libqt5quick5 libqt5quickwidgets5 qml-module-qtquick2 libgsettings-qt1
```

这些是搜狗拼音运行所需的 Qt 组件，缺少会导致输入法无法启动或候选框显示异常。

### 第四步：下载搜狗输入法

搜狗官方 CDN (`ime-sec.gtimg.com`) 封锁了非浏览器下载，需使用备用域名：

```bash
cd ~/Downloads
curl -L -o sogoupinyin_4.2.1.145_amd64.deb \
  -H "Referer: https://shurufa.sogou.com/" \
  "https://ime.gtimg.com/pc/dl/gzindex/1680521603/sogoupinyin_4.2.1.145_amd64.deb"
```

> **备用下载方式**：访问 [https://shurufa.sogou.com/linux](https://shurufa.sogou.com/linux)，点击"立即下载"，用浏览器下载。

**版本信息**：
- 包名: `sogoupinyin_4.2.1.145_amd64.deb`
- 大小: ~89 MB
- 架构: amd64 (x86_64)

### 第五步：安装搜狗输入法

```bash
sudo dpkg -i ~/Downloads/sogoupinyin_4.2.1.145_amd64.deb
sudo apt --fix-broken install
```

### 第六步：配置输入法系统

#### 6.1 切换输入法框架

1. 打开系统 **Settings** → **Region & Language** → **Manage Installed Languages**
2. 在 **"Keyboard input method system"** 下拉菜单中选择 **`fcitx`**
3. 点击 **"Apply System-Wide"**
4. 登出当前会话并重新登录（或重启）

#### 6.2 添加搜狗拼音到输入法列表

1. 重启后，右上角系统托盘找到 **小键盘图标**（fcitx）
2. 点击 → **Configure**
3. 点击左下角 **"+"** 按钮
4. 搜索 `Sogou Pinyin`，点击添加
5. 如果搜不到，取消勾选 **"Only Show Current Language"** 再试

#### 6.3 验证

- 打开任意文本编辑器（如 `gedit`）
- 按 `Ctrl + Space` 切换输入法
- 输入拼音测试 ✓

## 配置搜狗输入法

### 设置小鹤双拼

1. 右上角搜狗图标 → **设置属性**
2. 左侧找到 **"双拼"** 或 **"双拼布局"**
3. 在方案下拉菜单选择 **"小鹤双拼"**
4. 保存

### 其他可配置项

| 配置项 | 路径 | 说明 |
|--------|------|------|
| 外观 | 设置属性 → 外观 | 候选词字体、皮肤 |
| 词库 | 设置属性 → 词库 | 导入细胞词库 |
| 快捷键 | fcitx 配置工具 | 修改切换快捷键 |
| 输入习惯 | 设置属性 → 输入 | 候选词数量、模糊音等 |

## 常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 安装后找不到搜狗拼音 | 未添加到输入法列表 | 打开 fcitx 配置，取消"仅显示当前语言"，添加搜狗 |
| 按快捷键无法切换 | fcitx 服务未运行 / 快捷键冲突 | 终端执行 `fcitx -d` 手动启动；检查 fcitx 配置中的快捷键 |
| 输入法频繁崩溃 | 与 ibus 框架冲突 | `sudo apt purge ibus` 卸载 ibus，重启 |
| 候选词框闪烁或显示异常 | Wayland 与搜狗不兼容 | 切换到 X11 会话 |
| 出现 GTK/Qt 模块错误 | 核心依赖缺失 | 重新执行第三步的 apt install |
| 中文候选词乱码 | 字体缺失 | `sudo apt install fonts-wqy-zenhei` 安装中文字体 |

### 完全卸载

```bash
# 卸载搜狗
sudo dpkg -r sogoupinyin

# 恢复 ibus
sudo apt install ibus
im-config -n ibus

# 清理 fcitx（可选）
sudo apt purge fcitx fcitx-bin fcitx-table fcitx-config-gtk

# 重启
sudo reboot
```

## 技术细节

### 环境变量加载顺序

```
~/.xprofile  →  X11 会话启动时加载（推荐）
~/.profile   →  登录 shell 加载（可能过晚）
/etc/profile.d/*.sh  →  需要 sudo 权限
```

搜狗输入法需要在桌面环境启动前就设置好 `XMODIFIERS`，所以 `~/.xprofile` 是最佳位置。

### 搜狗输入法文件结构

```
/usr/share/fcitx/addon/        # 输入法插件
  fcitx-sogoupinyin.so        # 搜狗主插件
/usr/share/fcitx/conf/         # fcitx 配置
~/.config/fcitx/              # 用户 fcitx 配置
~/.config/SogouPY/            # 搜狗用户词库和配置
/usr/share/applications/sogoupinyin.desktop  # 桌面启动器
```

## 替代方案参考

如果搜狗输入法在当前环境下稳定性不佳，可考虑：

| 方案 | 安装难度 | 特点 |
|------|----------|------|
| **RIME / 中州韵** | ⭐⭐⭐ | 高度可定制、轻量、仍在活跃维护 |
| **Google 中文输入法 (ibus)** | ⭐ | 简单稳定，基于 ibus |
| **Fcitx5 + RIME** | ⭐⭐⭐ | 现代架构 + 可定制 |
| **Fcitx5 + PIME** | ⭐⭐ | Fcitx5 原生中文支持 |
